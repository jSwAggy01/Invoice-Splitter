Split multi-page PDFs into individually named files — runs entirely in your browser

Choose the multi-page PDF you want to split into individual pages.

Click to browse or drag & drop your PDF here

Select Names File

A text file (.txt) with one name per line. Each name becomes a file name for the corresponding page.

Click to browse or drag & drop your names file here
3

Choose a name for the folder your split PDFs will be placed in. When you extract the ZIP, everything will be inside this folder.
